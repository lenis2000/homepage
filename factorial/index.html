Exact sampler for the factorial Schur process

Reverse-Cauchy sampler for sλ(x | y) · šλ(w | y), implemented by the Yang--Baxter bijectivisation. The local random split is (x+y)/(w+y) versus (w−x)/(w+y).

About the model and exact sampler

The sampler starts from the frozen right-hand side configuration of the Cauchy identity

Σλ sλ(x1,…,xN | y) šλ(w1,…,wM | y) = Πi,j(wj−xi)−1,

and runs the reverse Cauchy sequence of local Yang--Baxter moves. In row language, the initially frozen order has all w-rows above all x-rows. The algorithm bubbles each x-row through the w-rows. Every adjacent row swap is deterministic except at one Bernoulli split with probabilities θ=(x+yk)/(w+yk) and 1−θ=(w−x)/(w+yk).

The output is the full two-sided Gelfand--Tsetlin path ensemble. The common boundary row is the sampled signature λ. For probabilistic parameters the page checks wj > xi for every pair before sampling.


Sizes & q-specialization

q-specialization sets xi = αqi, wk = γqk, yr = βqr. Choose parameters with every wk > xi; if samples stay at λ=0, increase the ratios xi/wk while preserving this inequality.

Spectral parameters x, w, y

Syntax: list 1,2,3 · repeat 0.5^4, 1^N, 1^M, 0^columnCap · pattern (0.5,0.7)^3 · expression alpha*q^i, 10*q^i, q^(-i)/2. Expressions are evaluated at i=1…N for x, i=1…M for w, and up to the column cap for y. Finite lists must be long enough; use a repeat when a constant sequence is intended.
