Glauber dynamics for the factorial Schur process

Two-sided GT pattern ∅ = λ⁽⁰⁾ ≺ λ⁽¹⁾ ≺ … ≺ λ⁽ᴺ⁾ = μ⁽ᴹ⁾ ≻ … ≻ μ⁽¹⁾ ≻ μ⁽⁰⁾ = ∅; single-flip Markov chain with Gibbs ratios from Theorem 1.12 of Yizhen Li's notes (May 2026).

About the model and dynamics

The state is a two-sided Gelfand–Tsetlin pattern (interlacing chain of partitions) of total height M+N: a chain rises from ∅ through λ⁽ⁱ⁾ in the x–y environment for i=0..N, and another rises from ∅ through μ⁽ʲ⁾ in the w–y environment for j=0..M; they meet at λ⁽ᴺ⁾=μ⁽ᴹ⁾.

The (un-normalized) weight is

W ∝ ∏j=1..M šμ⁽ʲ⁾/μ⁽ʲ⁻¹⁾(wj | y) ∙ ∏i=1..N sλ⁽ⁱ⁾/λ⁽ⁱ⁻¹⁾(xi | y).

Single-flip Glauber dynamics picks a flippable interior position uniformly and changes its value by ±1 according to the local Gibbs ratios:

  • μ-interior, level j ∈ [1, M−1], row i ∈ [1, N]: P(μ*)/P(μ) = (wj+1+yμᵢ+N+1−i) / (wj+yμᵢ+N+2−i)
  • λ-interior, level j ∈ [1, N−1], row i ∈ [1, j]: P(μ*)/P(μ) = (xj+yλᵢ+j+1−i) / (xj+1+yλᵢ+j+2−i)
  • middle row j=M (=j=N), row i: P(μ*)/P(μ) = (xN+yμᵢ+N+1−i) / (wM+yμᵢ+N+2−i)

The visualization shows N non-intersecting lattice paths in the strip; particle i at level k sits at column pi(k) determined by the partition at that level (μ side uses N-row convention so all N particles are visible; λ side uses j-row convention so particles "exit to the right" as we go up).

Source notes: Yizhen Li, "Factorial Schur" (2026), Definitions 1.7, 1.9, Theorem 1.12.

Sizes & q-specialization

q-specialization sets xi = α qi (i=1..N), wk = γ qk (k=1..M), yj = β qj (j=1..N+max λ).

Spectral parameters x, w, y

Syntax: 1,2,3, 1^4 = 1,1,1,1, (0.5,0.7)^3 = 0.5,0.7,0.5,0.7,0.5,0.7. The y array must be long enough to cover all particle positions.
