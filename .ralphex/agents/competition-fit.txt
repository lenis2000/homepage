## Judging Criteria Evaluation

# CREATIVITY (weight: high)
# - Is the concept genuinely novel? (Penrose triangle + probabilistic tiling + annealing)
# - Does it go beyond standard data visualization?
# - Is there intellectual depth beyond surface aesthetics?
# - Does it introduce ideas the viewer hasn't encountered?

# AESTHETICS (weight: high)
# - Does the visual treatment elevate the concept?
# - Is the color palette intentional and effective?
# - Is there visual sophistication in the rendering?
# - Does it look like it belongs in a gallery, not a CS lecture?

# STORYTELLING (weight: high)
# - Is there a clear narrative arc? (chaos -> emergence -> impossible truth)
# - Does the viewer understand the story without explanation?
# - Is the text (hook question, caption) effective?
# - Does the interactive element (entropy slider) enhance the story?

# INNOVATIVE INTEGRATION OF DATA (weight: high)
# - The "data" here is probability distributions on lozenge tilings
# - Is the connection between mathematical data and visual form clear?
# - Does the piece make a genuine statement about data/probability/truth?
# - Would a data science audience appreciate the mathematical depth?
