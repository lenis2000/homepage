## Evaluation Framework

# 1. First Impression (3-second test)
#    - Does the hook screen command attention instantly?
#    - Would a gallery visitor stop and engage?
#    - Does it read as "art" not "tech demo"?

# 2. Differentiation from 2024 Finalists
#    - Reef Generations: acoustic/ecological data
#    - Nebulae: particle simulation
#    - LA AQI: environmental time-series
#    - Melody of Budapest: geospatial music data
#    - This piece must stand apart through mathematical elegance + impossibility concept

# 3. Display Readiness
#    - Will it work on a wall-mounted display for months?
#    - Responsive across screen sizes?
#    - Does the animation loop or require interaction?
#    - Power/performance: can it run 24/7 without degrading?

# 4. Submission Description Readiness
#    - Can the piece be described in terms of: data source, visualization technique, narrative?
#    - "Data source" = probability distributions on lozenge tilings
#    - "Technique" = MCMC sampling, simulated annealing, impossible geometry
#    - "Narrative" = chaos to order, local truth vs global paradox

# 5. Polish Level
#    - Pixel-perfect rendering (no aliasing artifacts, seams, or rendering glitches)?
#    - Smooth animation (no jank, no stutter)?
#    - Professional typography?
#    - Does it feel finished, not a prototype?
