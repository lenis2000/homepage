In a recent paper Faraut [Tunisian J. Math. \textbf{1} (2019), 585--606] has observed that the corresponding eigenvalue PDF has an almost identical structure to the eigenvalue PDF for the rank 1 perturbation <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>A</mi><mo>+</mo><mi>b</mi><mover accent="true"><mi mathvariant="bold-italic">x</mi><mo>^</mo></mover><msup><mover accent="true"><mi mathvariant="bold-italic">x</mi><mo>^</mo></mover><mo>†</mo></msup></mrow><annotation encoding="application/x-tex">A + b \hat{\boldsymbol x} \hat{\boldsymbol x}^\dagger</annotation></semantics></math></span>, and asks for an explanation. We provide this by way of a common derivation involving the secular equations and associated Jacobians. This applies too in related setting, for example when <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mover accent="true"><mi mathvariant="bold-italic">x</mi><mo>^</mo></mover></mrow><annotation encoding="application/x-tex">\hat{\boldsymbol x}</annotation></semantics></math></span> is a real Gaussian and <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>A</mi></mrow><annotation encoding="application/x-tex">A</annotation></semantics></math></span> Hermitian, and also in a multiplicative setting <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>A</mi><mi>U</mi><mi>B</mi><msup><mi>U</mi><mo>†</mo></msup></mrow><annotation encoding="application/x-tex">A U B U^\dagger</annotation></semantics></math></span> where <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>A</mi><mo separator="true">,</mo><mi>B</mi></mrow><annotation encoding="application/x-tex">A, B</annotation></semantics></math></span> are fixed unitary matrices with <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>B</mi></mrow><annotation encoding="application/x-tex">B</annotation></semantics></math></span> a multiplicative rank 1 deviation from unity, and <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>U</mi></mrow><annotation encoding="application/x-tex">U</annotation></semantics></math></span> is a Haar distributed unitary matrix. Specifically, in each case there is a dual eigenvalue problem giving rise to a PDF of almost identical structure.</p>{% endraw %}
