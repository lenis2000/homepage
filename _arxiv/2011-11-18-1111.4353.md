{% raw %}<p>The problem of calculation of correlation functions in the six-vertex model with domain wall boundary conditions is addressed by considering a particular nonlocal correlation function, called row configuration probability. This correlation function can be used as building block for computing various (both local and nonlocal) correlation functions in the model. The row configuration probability is calculated using the quantum inverse scattering method; the final result is given in terms of a multiple integral. The connection with the emptiness formation probability, another nonlocal correlation function which was computed elsewhere using similar methods, is also discussed.</p>{% endraw %}
