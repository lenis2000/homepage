{% raw %}<p>The <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>τ</mi></mrow><annotation encoding="application/x-tex">\tau</annotation></semantics></math></span>-function theory of Painlev\'e systems is used to derive recurrences in the rank <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>n</mi></mrow><annotation encoding="application/x-tex">n</annotation></semantics></math></span> of certain random matrix averages over U(n). These recurrences involve auxilary quantities which satisfy discrete Painlev\'e equations. The random matrix averages include cases which can be interpreted as eigenvalue distributions at the hard edge and in the bulk of matrix ensembles with unitary symmetry. The recurrences are illustrated by computing the value of a sequence of these distributions as <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>n</mi></mrow><annotation encoding="application/x-tex">n</annotation></semantics></math></span> varies, and demonstrating convergence to the value of the appropriate limiting distribution.</p>{% endraw %}
