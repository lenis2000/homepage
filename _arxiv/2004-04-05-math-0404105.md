{% raw %}<p>We give potential theoretic estimates for the probability that a set <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>A</mi></mrow><annotation encoding="application/x-tex">A</annotation></semantics></math></span> contains a double point of planar Brownian motion run for unit time. Unlike the probability for <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>A</mi></mrow><annotation encoding="application/x-tex">A</annotation></semantics></math></span> to intersect the range of a Markov process, this cannot be estimated by a capacity of the set <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>A</mi></mrow><annotation encoding="application/x-tex">A</annotation></semantics></math></span>. Instead, we introduce the notion of a capacity with respect to two gauge functions simultaneously. We also give a polar decomposition of <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>A</mi></mrow><annotation encoding="application/x-tex">A</annotation></semantics></math></span> into a set that never intersects the set of Brownian double points and a set for which intersection with the set of Brownian double points is the same as intersection with the Brownian path.</p>{% endraw %}
