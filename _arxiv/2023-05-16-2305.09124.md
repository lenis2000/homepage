{% raw %}<p>Random matrices from the elliptic Ginibre orthogonal ensemble (GinOE) are a certain linear combination of a real symmetric, and real anti-symmetric, real Gaussian random matrices and controlled by a parameter <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>τ</mi></mrow><annotation encoding="application/x-tex">\tau</annotation></semantics></math></span>. Our interest is in the fluctuations of the number of real eigenvalues, for fixed <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>τ</mi></mrow><annotation encoding="application/x-tex">\tau</annotation></semantics></math></span> when the expected number is proportional to the square root of the matrix size <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>N</mi></mrow><annotation encoding="application/x-tex">N</annotation></semantics></math></span>, and for <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>τ</mi></mrow><annotation encoding="application/x-tex">\tau</annotation></semantics></math></span> scaled to the weakly non-symmetric limit, when the number of eigenvalues is proportional to <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>N</mi></mrow><annotation encoding="application/x-tex">N</annotation></semantics></math></span>. By establishing that the generating function for the probabilities specifying the distribution of the number of real eigenvalues has only negative real zeros, and using too the fact that variances in both circumstancesof interest tends to infinity as <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>N</mi><mo>→</mo><mi mathvariant="normal">∞</mi></mrow><annotation encoding="application/x-tex">N \to \infty</annotation></semantics></math></span>, the known central limit theorem for the fluctuations is strengthened to a local central limit theorem, and the rate of convergence is discussed.</p>{% endraw %}
