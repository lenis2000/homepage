{% raw %}<p>We present two new connections between the inhomogeneous stochastic higher spin six vertex model in a quadrant and integrable stochastic systems from the Macdonald processes hierarchy. First, we show how Macdonald <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>q</mi></mrow><annotation encoding="application/x-tex">q</annotation></semantics></math></span>-difference operators with <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>t</mi><mo>=</mo><mn>0</mn></mrow><annotation encoding="application/x-tex">t=0</annotation></semantics></math></span> (an algebraic tool crucial for studying the corresponding Macdonald processes) can be utilized to get <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>q</mi></mrow><annotation encoding="application/x-tex">q</annotation></semantics></math></span>-moments of the height function <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi mathvariant="fraktur">h</mi></mrow><annotation encoding="application/x-tex">\mathfrak{h}</annotation></semantics></math></span> in the higher spin six vertex model first computed in arXiv:1601.05770 using Bethe ansatz. This result in particular implies that for the vertex model with the step Bernoulli boundary condition, the value of <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi mathvariant="fraktur">h</mi></mrow><annotation encoding="application/x-tex">\mathfrak{h}</annotation></semantics></math></span> at an arbitrary point <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mo stretchy="false">(</mo><mi>N</mi><mo>+</mo><mn>1</mn><mo separator="true">,</mo><mi>T</mi><mo stretchy="false">)</mo><mo>∈</mo><msub><mi mathvariant="double-struck">Z</mi><mrow><mo>≥</mo><mn>2</mn></mrow></msub><mo>×</mo><msub><mi mathvariant="double-struck">Z</mi><mrow><mo>≥</mo><mn>1</mn></mrow></msub></mrow><annotation encoding="application/x-tex">(N+1,T)\in\mathbb{Z}_{\ge2}\times\mathbb{Z}_{\ge1}</annotation></semantics></math></span> has the same distribution as the last component <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><msub><mi>λ</mi><mi>N</mi></msub></mrow><annotation encoding="application/x-tex">\lambda_N</annotation></semantics></math></span> of a random partition under a specific <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>t</mi><mo>=</mo><mn>0</mn></mrow><annotation encoding="application/x-tex">t=0</annotation></semantics></math></span> Macdonald measure. On the other hand, it is known that <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><msub><mi mathvariant="bold">x</mi><mi>N</mi></msub><mo>:</mo><mo>=</mo><msub><mi>λ</mi><mi>N</mi></msub><mo>−</mo><mi>N</mi></mrow><annotation encoding="application/x-tex">\mathbf{x}_N:=\lambda_N-N</annotation></semantics></math></span> can be identified with the location of the <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>N</mi></mrow><annotation encoding="application/x-tex">N</annotation></semantics></math></span>th particle in a certain discrete time <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>q</mi></mrow><annotation encoding="application/x-tex">q</annotation></semantics></math></span>-TASEP started from the step initial configuration. The second construction we present is a coupling of this <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>q</mi></mrow><annotation encoding="application/x-tex">q</annotation></semantics></math></span>-TASEP and the higher spin six vertex model (with the step Bernoulli boundary condition) along time-like paths providing an independent probabilistic explanation of the equality of <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi mathvariant="fraktur">h</mi><mo stretchy="false">(</mo><mi>N</mi><mo>+</mo><mn>1</mn><mo separator="true">,</mo><mi>T</mi><mo stretchy="false">)</mo></mrow><annotation encoding="application/x-tex">\mathfrak{h}(N+1,T)</annotation></semantics></math></span> and <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><msub><mi mathvariant="bold">x</mi><mi>N</mi></msub><mo>+</mo><mi>N</mi></mrow><annotation encoding="application/x-tex">\mathbf{x}_N+N</annotation></semantics></math></span> in distribution. Combined with the identification of averages of observables between the stochastic higher spin six vertex model and Schur measures (which are <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>t</mi><mo>=</mo><mi>q</mi></mrow><annotation encoding="application/x-tex">t=q</annotation></semantics></math></span> Macdonald measures) obtained recently in arXiv:1608.01553, this produces GUE Tracy--Widom asymptotics for a discrete time <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>q</mi></mrow><annotation encoding="application/x-tex">q</annotation></semantics></math></span>-TASEP with the step initial configuration and special jump parameters.</p>{% endraw %}
