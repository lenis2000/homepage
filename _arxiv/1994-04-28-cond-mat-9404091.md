{% raw %}<p>We consider the probability to find a string of <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>x</mi></mrow><annotation encoding="application/x-tex">x</annotation></semantics></math></span> adjacent parallel spins in the antiferromagnetic ground state of the model (in a magnetic field). We derive a system of integro-difference equations which define this probability. This system is completely integrable, it has Lax representation and a corresponding Riemann-Hilbert problem. The quantum correlation funtion is a <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>τ</mi></mrow><annotation encoding="application/x-tex">\tau</annotation></semantics></math></span>-function of this system.</p>{% endraw %}
