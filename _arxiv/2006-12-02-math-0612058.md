{% raw %}<p>In this work we study the Plancherel-Rotach type asymptotics for <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>q</mi></mrow><annotation encoding="application/x-tex">q</annotation></semantics></math></span>-Laguerre orthogonal polynomials with complex scaling . The main term of the asymptotics contains Ramanujan function <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><msub><mi>A</mi><mi>q</mi></msub><mo stretchy="false">(</mo><mi>z</mi><mo stretchy="false">)</mo></mrow><annotation encoding="application/x-tex">A_{q}(z)</annotation></semantics></math></span> for the scaling parameter on the vertical line <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi mathvariant="normal">ℜ</mi><mo stretchy="false">(</mo><mi>s</mi><mo stretchy="false">)</mo><mo>=</mo><mn>2</mn></mrow><annotation encoding="application/x-tex">\Re(s)=2</annotation></semantics></math></span>, while the main term of the asymptotics involves the theta function for the scaling parameter in the strip <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mn>0</mn><mo>&lt;</mo><mi mathvariant="normal">ℜ</mi><mo stretchy="false">(</mo><mi>s</mi><mo stretchy="false">)</mo><mo>&lt;</mo><mn>2</mn></mrow><annotation encoding="application/x-tex">0&lt;\Re(s)&lt;2</annotation></semantics></math></span>. In the latter case the number theoretical property of the scaling parameter completely determines the order of the error term. <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow></mrow><annotation encoding="application/x-tex"> </annotation></semantics></math></span>These asymptotic formulas may provide insights to some new random matrix model and add a new link between special functions and number theory.</p>{% endraw %}
