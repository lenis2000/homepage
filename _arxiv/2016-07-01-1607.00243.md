{% raw %}<p>In this paper, we investigate the extremal values of (the logarithm of) the characteristic polynomial of a random unitary matrix whose spectrum is distributed according the Circular Beta Ensemble (C<span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>β</mi></mrow><annotation encoding="application/x-tex">\beta</annotation></semantics></math></span>E). More precisely, if <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><msub><mi>X</mi><mi>n</mi></msub></mrow><annotation encoding="application/x-tex">X_n</annotation></semantics></math></span> is this characteristic polynomial and <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi mathvariant="double-struck">U</mi></mrow><annotation encoding="application/x-tex">\mathbb{U}</annotation></semantics></math></span> the unit circle, we prove that: <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML" display="block"><semantics><mrow><munder><mrow><mi>sup</mi><mo>⁡</mo></mrow><mrow><mi>z</mi><mo>∈</mo><mi mathvariant="double-struck">U</mi></mrow></munder><mi mathvariant="normal">ℜ</mi><mi>log</mi><mo>⁡</mo><msub><mi>X</mi><mi>n</mi></msub><mo stretchy="false">(</mo><mi>z</mi><mo stretchy="false">)</mo><mo>=</mo><msqrt><mfrac><mn>2</mn><mi>β</mi></mfrac></msqrt><mrow><mo fence="true">(</mo><mi>log</mi><mo>⁡</mo><mi>n</mi><mo>−</mo><mfrac><mn>3</mn><mn>4</mn></mfrac><mi>log</mi><mo>⁡</mo><mi>log</mi><mo>⁡</mo><mi>n</mi><mo>+</mo><mi mathvariant="script">O</mi><mo stretchy="false">(</mo><mn>1</mn><mo stretchy="false">)</mo><mo fence="true">)</mo></mrow><mtext> </mtext><mo separator="true">,</mo></mrow><annotation encoding="application/x-tex">\sup_{z \in \mathbb{U} } \Re \log X_n(z) = \sqrt{\frac{2}{\beta}} \left(\log n - \frac{3}{4} \log \log n + \mathcal{O}(1) \right)\ ,</annotation></semantics></math></span> as well as an analogous statement for the imaginary part. The notation <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi mathvariant="script">O</mi><mo stretchy="false">(</mo><mn>1</mn><mo stretchy="false">)</mo></mrow><annotation encoding="application/x-tex">\mathcal{O}(1)</annotation></semantics></math></span> means that the corresponding family of random variables, indexed by <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>n</mi></mrow><annotation encoding="application/x-tex">n</annotation></semantics></math></span>, is tight. This answers a conjecture of Fyodorov, Hiary and Keating, originally formulated for the case where <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>β</mi></mrow><annotation encoding="application/x-tex">\beta</annotation></semantics></math></span> equals to <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mn>2</mn></mrow><annotation encoding="application/x-tex">2</annotation></semantics></math></span>, which corresponds to the CUE field.</p>{% endraw %}
