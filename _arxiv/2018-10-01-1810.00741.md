{% raw %}<p>The Muttalib-Borodin biorthogonal ensemble is a probability density function for <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>n</mi></mrow><annotation encoding="application/x-tex">n</annotation></semantics></math></span> particles on the positive real line that depends on a parameter <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>θ</mi></mrow><annotation encoding="application/x-tex">\theta</annotation></semantics></math></span> and an external field <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>V</mi></mrow><annotation encoding="application/x-tex">V</annotation></semantics></math></span>. For <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>θ</mi><mo>=</mo><mfrac><mn>1</mn><mn>2</mn></mfrac></mrow><annotation encoding="application/x-tex">\theta=\frac{1}{2}</annotation></semantics></math></span> we find the large <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>n</mi></mrow><annotation encoding="application/x-tex">n</annotation></semantics></math></span> behavior of the associated correlation kernel with only few restrictions on <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>V</mi></mrow><annotation encoding="application/x-tex">V</annotation></semantics></math></span>. The idea is to relate the ensemble to a type II multiple orthogonal polynomial ensemble that can in turn be related to a <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mn>3</mn><mo>×</mo><mn>3</mn></mrow><annotation encoding="application/x-tex">3\times 3</annotation></semantics></math></span> Riemann-Hilbert problem which we then solve with the Deift-Zhou steepest descent method. The main ingredient is the construction of the local parametrix at the origin, with the help of Meijer G-functions, and its matching condition with a global parametrix. We will present a new iterative technique to obtain the matching condition, which we expect to be applicable in more general situations as well.</p>{% endraw %}
