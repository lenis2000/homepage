{% raw %}<p>A conserved generalized zero range process is considered in which two sites interact such that particles hop from the more populated site to the other with a probability <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>p</mi></mrow><annotation encoding="application/x-tex">p</annotation></semantics></math></span>. The steady state particle distribution function <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>P</mi><mo stretchy="false">(</mo><mi>n</mi><mo stretchy="false">)</mo></mrow><annotation encoding="application/x-tex">P(n)</annotation></semantics></math></span> is obtained using both analytical and numerical methods. The system goes through several phases as <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>p</mi></mrow><annotation encoding="application/x-tex">p</annotation></semantics></math></span> is varied. In particular, a condensate phase appears for <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><msub><mi>p</mi><mi>l</mi></msub><mo>&lt;</mo><mi>p</mi><mo>&lt;</mo><msub><mi>p</mi><mi>c</mi></msub></mrow><annotation encoding="application/x-tex">p_l &lt; p &lt; p_c</annotation></semantics></math></span>, where the bounding values depend on the range of interaction, with <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><msub><mi>p</mi><mi>c</mi></msub><mo>&lt;</mo><mn>0.5</mn></mrow><annotation encoding="application/x-tex">p_c &lt; 0.5</annotation></semantics></math></span> in general. Analysis of <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>P</mi><mo stretchy="false">(</mo><mi>n</mi><mo stretchy="false">)</mo></mrow><annotation encoding="application/x-tex">P(n)</annotation></semantics></math></span> in the condensate phase using a known scaling form shows there is universal behaviour in the short range process while the infinite range process displays non-universality. In the non-condensate phase above <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><msub><mi>p</mi><mi>c</mi></msub></mrow><annotation encoding="application/x-tex">p_c</annotation></semantics></math></span>, two distinct regions are identified: <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><msub><mi>p</mi><mi>c</mi></msub><mo>&lt;</mo><mi>p</mi><mo>≤</mo><mn>0.5</mn></mrow><annotation encoding="application/x-tex">p_c &lt; p \leq 0.5</annotation></semantics></math></span> and <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>p</mi><mo>&gt;</mo><mn>0.5</mn></mrow><annotation encoding="application/x-tex">p&gt; 0.5</annotation></semantics></math></span>; a scale emerges in the system in the latter and this feature is present for all ranges of interaction.</p>{% endraw %}
