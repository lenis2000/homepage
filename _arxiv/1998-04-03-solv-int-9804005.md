{% raw %}<p>For the unitary ensembles of <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>N</mi><mo>×</mo><mi>N</mi></mrow><annotation encoding="application/x-tex">N\times N</annotation></semantics></math></span> Hermitian matrices associated with a weight function <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>w</mi></mrow><annotation encoding="application/x-tex">w</annotation></semantics></math></span> there is a kernel, expressible in terms of the polynomials orthogonal with respect to the weight function, which plays an important role. For the orthogonal and symplectic ensembles of Hermitian matrices there are <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mn>2</mn><mo>×</mo><mn>2</mn></mrow><annotation encoding="application/x-tex">2\times2</annotation></semantics></math></span> matrix kernels, usually constructed using skew-orthogonal polynomials, which play an analogous role. These matrix kernels are determined by their upper left-hand entries. We derive formulas expressing these entries in terms of the scalar kernel for the corresponding unitary ensembles. We also show that whenever <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><msup><mi>w</mi><mo mathvariant="normal" lspace="0em" rspace="0em">′</mo></msup><mi mathvariant="normal">/</mi><mi>w</mi></mrow><annotation encoding="application/x-tex">w&#x27;/w</annotation></semantics></math></span> is a rational function the entries are equal to the scalar kernel plus some extra terms whose number equals the order of <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><msup><mi>w</mi><mo mathvariant="normal" lspace="0em" rspace="0em">′</mo></msup><mi mathvariant="normal">/</mi><mi>w</mi></mrow><annotation encoding="application/x-tex">w&#x27;/w</annotation></semantics></math></span>. General formulas are obtained for these extra terms. We do not use skew-orthogonal polynomials in the derivations.</p>{% endraw %}
