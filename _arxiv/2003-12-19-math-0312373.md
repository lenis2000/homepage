{% raw %}<p>The shifted Schur measure introduced by Tracy and Widom is a measure on the set of all strict partitions, which is defined by Schur <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>Q</mi></mrow><annotation encoding="application/x-tex">Q</annotation></semantics></math></span>-functions. The main aim of this paper is to calculate the correlation function of this measure, which is given by a pfaffian. As an application, we prove that a limit distribution of <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><msub><mi>λ</mi><mi>j</mi></msub></mrow><annotation encoding="application/x-tex">\lambda_j</annotation></semantics></math></span>'s with respect to a shifted version of the Plancherel measure for symmetric groups is identical with the corresponding distribution of the original Plancherel measure. Further we give expressions of the mean value and the variance of the size of a partition with respect to the measure defined by Hall-Littlewood functions.</p>{% endraw %}
