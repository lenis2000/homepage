{% raw %}<p>We consider a generalization of the Schur process in which a partition evolves from the empty partition into an arbitrary fixed final partition. We obtain a double integral representation of the correlation kernel. For a special final partition with only one row, the edge scaling limit is also discussed by the use of the saddle point analysis. If we appropriately scale the length of the row, the limiting correlation kernel changes from the extended Airy kernel.</p>{% endraw %}
