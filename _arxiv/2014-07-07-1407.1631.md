{% raw %}<p>We consider an exclusion process on a ring in which a particle hops to an empty neighbouring site with a rate that depends on the number of vacancies <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>n</mi></mrow><annotation encoding="application/x-tex">n</annotation></semantics></math></span> in front of it. In the steady state, using the well known mapping of this model to the zero range process, we write down an exact formula for the partition function and the particle-particle correlation function in the canonical ensemble. In the thermodynamic limit, we find a simple analytical expression for the generating function of the correlation function. This result is applied to the hop rate <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>u</mi><mo stretchy="false">(</mo><mi>n</mi><mo stretchy="false">)</mo><mo>=</mo><mn>1</mn><mo>+</mo><mo stretchy="false">(</mo><mi>b</mi><mi mathvariant="normal">/</mi><mi>n</mi><mo stretchy="false">)</mo></mrow><annotation encoding="application/x-tex">u(n)=1+(b/n)</annotation></semantics></math></span> for which a phase transition between high-density laminar phase and low-density jammed phase occurs for <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>b</mi><mo>&gt;</mo><mn>2</mn></mrow><annotation encoding="application/x-tex">b &gt; 2</annotation></semantics></math></span>. For these rates, we find that at the critical density, the correlation function decays algebraically with a continuously varying exponent <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>b</mi><mo>−</mo><mn>2</mn></mrow><annotation encoding="application/x-tex">b-2</annotation></semantics></math></span>. We also calculate the two-point correlation function above the critical density, and find that the correlation length diverges with a critical exponent <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>ν</mi><mo>=</mo><mn>1</mn><mi mathvariant="normal">/</mi><mo stretchy="false">(</mo><mi>b</mi><mo>−</mo><mn>2</mn><mo stretchy="false">)</mo></mrow><annotation encoding="application/x-tex">\nu=1/(b-2)</annotation></semantics></math></span> for <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>b</mi><mo>&lt;</mo><mn>3</mn></mrow><annotation encoding="application/x-tex">b &lt; 3</annotation></semantics></math></span> and <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mn>1</mn></mrow><annotation encoding="application/x-tex">1</annotation></semantics></math></span> for <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>b</mi><mo>&gt;</mo><mn>3</mn></mrow><annotation encoding="application/x-tex">b &gt; 3</annotation></semantics></math></span>. These results are compared with those obtained using an exact series expansion for finite systems.</p>{% endraw %}
