{% raw %}<p>We prove that in the full <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><msup><mi>L</mi><mn>2</mn></msup></mrow><annotation encoding="application/x-tex">L^2</annotation></semantics></math></span>-regime the partition function of the directed polymer model in dimensions <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>d</mi><mo>≥</mo><mn>3</mn></mrow><annotation encoding="application/x-tex">d\geq 3</annotation></semantics></math></span>, if centered, scaled and averaged with respect to a test function <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>φ</mi><mo>∈</mo><msub><mi>C</mi><mi>c</mi></msub><mo stretchy="false">(</mo><msup><mi mathvariant="double-struck">R</mi><mi>d</mi></msup><mo stretchy="false">)</mo></mrow><annotation encoding="application/x-tex">\varphi \in C_c(\mathbb{R}^d)</annotation></semantics></math></span>, converges in distribution to a Gaussian random variable with explicit variance. Introducing a new idea in this context of a martingale difference representation, we also prove that the log-partition function, which can be viewed as a discretisation of the KPZ equation, exhibits the same fluctuations, when centered and averaged with respect to a test function. Thus, the two models fall within the Edwards-Wilkinson universality class in the full <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><msup><mi>L</mi><mn>2</mn></msup></mrow><annotation encoding="application/x-tex">L^2</annotation></semantics></math></span>-regime, a result that was only established, so far, for a strict subset of this regime in <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>d</mi><mo>≥</mo><mn>3</mn></mrow><annotation encoding="application/x-tex">d\geq 3</annotation></semantics></math></span>.</p>{% endraw %}
