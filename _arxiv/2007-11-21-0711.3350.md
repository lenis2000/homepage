{% raw %}<p>This paper examines how close the chordal $\SLE_\kappa$ curve gets to the real line asymptotically far away from its starting point. In particular, when <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>κ</mi><mo>∈</mo><mo stretchy="false">(</mo><mn>0</mn><mo separator="true">,</mo><mn>4</mn><mo stretchy="false">)</mo></mrow><annotation encoding="application/x-tex">\kappa\in(0,4)</annotation></semantics></math></span>, it is shown that if <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>β</mi><mo>&gt;</mo><msub><mi>β</mi><mi>κ</mi></msub><mo>:</mo><mo>=</mo><mn>1</mn><mi mathvariant="normal">/</mi><mo stretchy="false">(</mo><mn>8</mn><mi mathvariant="normal">/</mi><mi>κ</mi><mo>−</mo><mn>2</mn><mo stretchy="false">)</mo></mrow><annotation encoding="application/x-tex">\beta&gt;\beta_\kappa:=1/(8/\kappa-2)</annotation></semantics></math></span>, then the intersection of the $\SLE_\kappa$ curve with the graph of the function <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>y</mi><mo>=</mo><mi>x</mi><mi mathvariant="normal">/</mi><mo stretchy="false">(</mo><mi>log</mi><mo>⁡</mo><mi>x</mi><msup><mo stretchy="false">)</mo><mi>β</mi></msup></mrow><annotation encoding="application/x-tex">y=x/(\log x)^{\beta}</annotation></semantics></math></span>, <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>x</mi><mo>&gt;</mo><mi>e</mi></mrow><annotation encoding="application/x-tex">x&gt;e</annotation></semantics></math></span>, is a.s. bounded, while it is a.s. unbounded if <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>β</mi><mo>=</mo><msub><mi>β</mi><mi>κ</mi></msub></mrow><annotation encoding="application/x-tex">\beta=\beta_\kappa</annotation></semantics></math></span>. The critical $\SLE_4$ curve a.s. intersects the graph of <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>y</mi><mo>=</mo><msup><mi>x</mi><mrow><mo>−</mo><mo stretchy="false">(</mo><mi>log</mi><mo>⁡</mo><mi>log</mi><mo>⁡</mo><mi>x</mi><msup><mo stretchy="false">)</mo><mi>α</mi></msup></mrow></msup></mrow><annotation encoding="application/x-tex">y=x^{-(\log\log x)^\alpha}</annotation></semantics></math></span>, <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>x</mi><mo>&gt;</mo><msup><mi>e</mi><mi>e</mi></msup></mrow><annotation encoding="application/x-tex">x&gt;e^e</annotation></semantics></math></span>, in an unbounded set if <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>α</mi><mo>≤</mo><mn>1</mn></mrow><annotation encoding="application/x-tex">\alpha\le 1</annotation></semantics></math></span>, but not if <span class="katex"><math xmlns="http://www.w3.org/1998/Math/MathML"><semantics><mrow><mi>α</mi><mo>&gt;</mo><mn>1</mn></mrow><annotation encoding="application/x-tex">\alpha&gt;1</annotation></semantics></math></span>.
