One-Periodic Random Edge Weights [Bufetov–P.–Zografos 2025]

Model:
Attach i.i.d. random weights \(\alpha_k\) to edges in each diagonal layer \(k = 1, \ldots, M\) of the Aztec diamond of size \(M\).
\(\displaystyle \mathbb{P}(\text{tiling}) = \frac{1}{Z(\alpha)} \prod_{\text{edges } e} \alpha_{k(e)}\)
One-periodic structure:
Weights depend only on the diagonal, so alternating layers have the same distribution (one-periodic in the diagonal direction)
Two variance regimes:
  1. Critical scaling: \(\operatorname{Var}(\alpha_k) \sim 1/M\)
  2. Fixed distribution: \(\operatorname{Var}(\alpha_k) = \text{const}\)
Key question:
How do random weights affect the limit shape and fluctuations?
