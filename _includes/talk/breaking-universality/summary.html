Summary

  • Universality well-developed, including local translation-invariant Gibbs measures, GFF global fluctuations (also Airy/KPZ at the edge, and several other universal regimes)
  • The waterfall/barcode phenomenon [Knizel–P. 2025] describes an instance of dimensional collapse in the local behavior. So far, isolated in the q-Racah hexagon tilings, but should be present more broadly
  • Random edge weights (disordered dimer models) is a new rich world with many new instances of potentially universal asymptotic behavior [Bufetov–P.–Zografos 2025], [Moulard–Toninelli 2025], [Zografos 2025], [Duits–Van Peski 2025]. Diagonally layered disorder is the most developed at this moment.
  • Results presented today result on specifics of breaking universality while preserving some exact solvability, so there is hope of rigorous analysis
