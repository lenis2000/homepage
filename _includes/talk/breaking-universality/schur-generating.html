Schur Generating Functions

Exact solvability:
Domino tilings of the Aztec diamond are governed by Schur processes [Johansson 2002], [Okounkov–Reshetikhin 2003]
Annealed Schur generating function:
For random edge weights \(\alpha_k\), average the partition function over the disorder:
\(\displaystyle \mathbb{E}_{\alpha}\left[\prod_k s_{\lambda^{(k)}/\lambda^{(k-1)}}(\alpha_k, \alpha_k^{-1})\right]\)
Key idea:
The annealed (disorder-averaged) correlation functions can be computed explicitly via Schur generating functions, yielding a determinantal point process with an explicit correlation kernel
Extensions [Bufetov–P.–Zografos 2025]:
We substantially extend the LLN and CLT for particle systems via Schur generating functions to cover the random environment setting
