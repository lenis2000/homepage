q-Racah Measure on Lozenge Tilings

Lozenges ↔ noncolliding paths \(x_1(t) < x_2(t) < \cdots < x_N(t)\)

Weight function:
\(\displaystyle \mathrm{w}_{q,\kappa}(j) = \kappa q^{j-(S+1)/2} - \frac{1}{\kappa q^{j-(S+1)/2}}\)
Probability measure:
\(\displaystyle \mathbb{P}(\text{tiling}) = \frac{1}{Z} \prod_{\text{horiz. lozenges}} \mathrm{w}_{q,\kappa}(\text{height})\)
Key property:
As \(\kappa \to 0\) or \(\kappa \to +\infty\):
q-Racah interpolates between \(q^{+\text{vol}}\) and \(q^{-\text{vol}}\)
Orthogonal polynomials hierarchy:
Uniform tilings ↔ Hahn polynomials → q-Racah measure ↔ q-Racah polynomials
References: [Borodin-Gorin-Rains 2009], [Gorin-Huang 2022], [Morales-Pak-Panova]
